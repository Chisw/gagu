## FileExplorer

| ![](/assets/icons/app-file-explorer.svg) FileExplorer | Mac | Win |

| Go forward | ⌘ + → | Ctrl + → |

| Select all | ⌘ + A | Ctrl + A |

| Copy | ⌘ + C | Ctrl + C |
| Cut | ⌘ + X | Ctrl + X |
| Paste | ⌘ + V | Ctrl + V |
| Download | ⌘ + D | Ctrl + D |
| Upload | ⌘ + U | Ctrl + U |
| Filter | ⌘ + F | Ctrl + F |
| Exit (Filter) | Esc | Esc |
| List view | ⌘ + L | Ctrl + L |
| Grid view | ⌘ + G | Ctrl + G |
| Switch hidden entries | ⌘ + H | Ctrl + H |
| Rename | Enter | F2 |
| Delete | ⌘ + Backspace | Shift + Del |

</div>
