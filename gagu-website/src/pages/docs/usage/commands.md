# Commands

Start service

```sh
$ gagu
```

Show help info

```sh
$ gagu -h
```

Start and open in browser

```sh
$ gagu -o
```

Start with customized port

```sh
$ gagu -p 8888
```

Show version

```sh
$ gagu -v
```

Reset with removing GAGU data directory

```sh
$ gagu --reset
```

Stop service

```sh
# Ctrl + C
```
