## Basic Usage

```sh
# Start service
$ gagu

# Show help info
$ gagu -h

# Start and open in browser
$ gagu -o

# Start with customized port
$ gagu -p 8888

# Show version
$ gagu -v

# Reset with removing GAGU root directory
$ gagu --reset

# Stop service
# Ctrl + C
```
